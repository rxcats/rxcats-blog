# MacOS

## Mac Xcode Command Line Toos 설치
```bash
xcode-select --install
```

## Oh My ZSH 설치
- https://ohmyz.sh/

```bash
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

# zsh-autosuggestions
git clone https://github.com/zsh-users/zsh-autosuggestions ${ZSH_CUSTOM}/plugins/zsh-autosuggestions

# zsh-syntax-highlighting
git clone https://github.com/zsh-users/zsh-syntax-highlighting.git ${ZSH_CUSTOM}/plugins/zsh-syntax-highlighting
```

## 패키지 관리자 Homebrew 설치
- https://brew.sh/

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

### Homebrew 환경설정
- Homebrew 설치후 $PATH 설정이 되지 않았을 경우 brew 명령어가 동작하지 않을 수 있으므로 $PATH 설정을 추가
- .zprofile 에 추가되도록 하기

```shell
echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile
source ~/.zprofile
brew doctor
```

## 패키지 관리자 asdf 설치
- https://asdf-vm.com/
- 예를들면 jdk 여러개의 버전을 설치하고 관리할 수 있음

```bash
brew install asdf
```

### JDK 설치
```shell
asdf plugin add java
asdf install java liberica-21.0.2+14
asdf global java liberica-21.0.2+14
```

### Node JS 설치

```bash
asdf plugin add nodejs
asdf install nodejs 20.11.1
asdf global nodejs 20.11.1
```

## Docker 설치
- Docker Desktop 대체
